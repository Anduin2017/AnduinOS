Diviértase con sus fotos

Shotwell es un administrador de fotos útil que está listo para sus dispositivos. Conecte una cámara o un teléfono para transferir sus fotos, luego es fácil compartirlas y mantenerlas seguras. Y si se siente creativo, puede encontrar muchas otras aplicaciones de fotos en el software de AnduinOS.

Software incluido

  • Gestor de fotos Shotwell

Software compatible

  • Editor de imágenes GIMP

  • Editor de video Shotcut
