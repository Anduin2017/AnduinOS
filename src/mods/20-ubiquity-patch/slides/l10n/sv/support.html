Hjälp och Support

Den officiella dokumentationen för AnduinOS täcker ett brett spektrum av problem du kan stöta på. Dessutom är AnduinOS community-forum fyllda med användare och utvecklare.

På Ask Ubuntu kan du söka efter och fråga om mycket vanliga problem. Lösningar för nästan alla AnduinOS-problem är identiska med de för Ubuntu. Du kan helt förlita dig på Ubuntus dokumentation och expertis för att lösa problem med AnduinOS.
