Tudo o que precisa para o escritório

LibreOffice é uma suite de produtividade gratuita com tudo o que precisa para criar documentos, folhas de cálculo e apresentações. Totalmente compatível com os formatos do Microsoft Office, dá-lhe todas as funcionalidades que precisa, mas sem um custo associado.

Software incluído

  • LibreOffice Writer

  • LibreOffice Calc

  • LibreOffice Impress
